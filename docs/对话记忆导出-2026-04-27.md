- [figures/real_characteristics/characteristic_summary.csv](/Users/Pang_chip/Documents/数据挖掘/figures/real_characteristics/characteristic_summary.csv)

按以下特征做了 low / medium / high 分组：

- anomaly_ratio
- mean_anomaly_duration
- seasonality_proxy
- noise_proxy

### 16.1 anomaly ratio

观察包括：

- low anomaly ratio 下，kNN 最强
- medium anomaly ratio 下，kNN 与 Isolation Forest 接近
- high anomaly ratio 下，Isolation Forest 更稳定

### 16.2 mean anomaly duration

观察包括：

- 短、中等异常持续长度下，kNN 较强
- 很长异常段下，整体都更难，Isolation Forest 相对稳一些

### 16.3 seasonality

观察包括：

- 周期性更强时，多种方法差距缩小
- Matrix Profile 在某些更有结构的时间序列上有提升空间

### 16.4 noise proxy

观察包括：

- 中等噪声下，kNN / LOF / Matrix Profile 都可能有较好表现
- 高噪声下，Isolation Forest 的稳健性更突出

然后生成了：

- [figures/real_ablation](/Users/Pang_chip/Documents/数据挖掘/figures/real_ablation)

### 17.2 normalization effect

对话中已经根据真实结果明确总结：

- `kNN` 受 normalization 影响最明显，z-normalization 后效果提升
- `Isolation Forest` 基本不受影响
- `LOF`、`IK-density`、`Matrix Profile` 变化很小

### 17.3 window size effect

后续为了避免原始消融表中不同数据集 window 长度混在一起，又额外按：

- `small_1pct`
- `default`
- `large_5pct`

做了聚合。

聚合后的结论包括：

- `default` 通常最稳
- `large_5pct` 会明显拉低 kNN 和 Isolation Forest 的表现
- `LOF` 在 large_5pct 下有时略有提升
- `Matrix Profile` 的表现与窗口大小高度相关
